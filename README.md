> Open-Source Desktop application for Graphite. (Practically wraps the web-app for now)

## Install

```sh
pnpm install
```

## Usage

```sh
pnpm run dev:native
```

## Run executable locally

```sh
pnpm tauri build && pnpm run preview:native-[win/linux]
```
